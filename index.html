[Arrays] 128.Longest Consecutive Sequence

15 ноября 2023 г.

Задача: на вход подается не отсортированный массив целых чисел nums, нужно вернуть длинну самой длинной растушей последовательности чисел.
Необходимо написать алгоритм срабатывающий за время O(n).
Ограничения:
0 <= nums.length <= 105
-109 <= nums[i] <= 109

[Arrays] 238.Product of Array Except Self

15 ноября 2023 г.

Задача: на вход подается массив целых чисел nums, нужно вернуть такой массив answer, в котором бы answer[i] будет равен произведению всех элементов массива nums, заисключением элемента nums[i].
Необходимо написать алгоритм срабатывающий за время O(n).
Ограничения:
2 <= nums.length <= 105
-30 <= nums[i] <= 30.
